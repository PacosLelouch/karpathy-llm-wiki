# Security Policy

## 支持范围

当前仓库处于早期版本，默认只维护最新版本。

## 报告安全问题

如果发现可能导致敏感资料泄露、误修改 `raw/`、越权写入、危险命令执行或错误归档私密内容的问题，请不要公开贴出敏感样例。请在私有渠道提供：

- 影响范围
- 复现步骤
- 相关文件路径
- 期望行为与实际行为

## 内容安全原则

- 不提交真实用户私密资料。
- 不提交访问令牌、Cookie、API Key、内部链接或受限文档。
- `raw/` 是目标 wiki 的资料层，不属于本 skill 仓库示例内容；示例必须使用虚构或公开可复用内容。
- 任何自动修复脚本都不得删除用户资料或重写原始来源。
